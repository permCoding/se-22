Во время парсинга с паджинацией между страницами сайта желательно делать небольшую паузу:  

```js
// возможный вариант функции для организации паузы

function delay(sek) {
    const date = Date.now();
    let cur = null;
    do { cur = Date.now(); } while (cur-date < sek * 1000);
}

delay(1.5) // в секундах

```
